# How to Navigate to Cloud Run in Google Cloud Console

## Step-by-Step Navigation

### Method 1: Direct URL (Fastest)

1. **Open your browser** and go to:
   ```
   https://console.cloud.google.com/run?project=studio-9165758963-a10e4
   ```

2. This will take you directly to the Cloud Run page for your project.

### Method 2: Manual Navigation

1. **Go to Google Cloud Console**
   - Visit: https://console.cloud.google.com/
   - Make sure you're logged in with: `piyushraj3315@gmail.com`

2. **Select Your Project**
   - At the top of the page, click the project dropdown (it might say "Select a project" or show another project name)
   - Search for or select: `studio-9165758963-a10e4`
   - Click on it to select

3. **Open the Navigation Menu**
   - Look for the **☰ (hamburger menu)** icon in the top-left corner
   - Click it to open the side navigation menu

4. **Find Cloud Run**
   - Scroll down in the menu
   - Look for **"Cloud Run"** under the **"Serverless"** section
   - Or use the search box at the top of the menu and type "Cloud Run"
   - Click on **"Cloud Run"**

5. **Create Service**
   - Once on the Cloud Run page, you'll see a button at the top that says **"+ CREATE SERVICE"** or **"CREATE SERVICE"**
   - Click it to start creating a new service

## Alternative: Search Method

1. Click the **☰ menu** icon
2. In the search box at the top of the menu, type: **"Cloud Run"**
3. Click on **"Cloud Run"** from the search results
4. Click **"+ CREATE SERVICE"** button

## If You Don't See Cloud Run

If Cloud Run is not visible in the menu, you may need to:

1. **Enable the Cloud Run API first:**
   - Go to: https://console.cloud.google.com/apis/library/run.googleapis.com?project=studio-9165758963-a10e4
   - Click **"Enable"** button
   - Wait for it to enable (takes a few seconds)
   - Then go back to Cloud Run

2. **Or use the direct API enablement:**
   - Go to: https://console.cloud.google.com/apis/library?project=studio-9165758963-a10e4
   - Search for "Cloud Run API"
   - Click on it
   - Click **"Enable"**
